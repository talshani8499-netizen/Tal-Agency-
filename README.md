# Elevate Digital

AI Automation Agency website for US small businesses. Built with React, Vite, Tailwind CSS, and React Router.

## Services
- AI Voice Agents
- AI Chat Agents
- AI-Powered Landing Pages
- Custom AI Automation

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Run the app: `npm run dev`
3. Open http://localhost:3000

## Build

```bash
npm run build
```
